# alf.mx

The development of my website, available on [https://alf.mx](https://alf.mx).

Feel free to dive in, learn from it, and copy code (MIT licensed). 

The content itself is not open source and remains protected. To give you the opportunity to install a working project, I made it all available on GitHub. Please don't misuse or redistribute the content. The code is open source, the creative work is not.

If you want, install the project locally and explore how it works.

## You need

You need a local dev environment with PHP 8.4 or higher, Node.js and MySQL. I recommend [Laravel Herd](https://herd.laravel.com/), it's free.

## Tech

I use the best of several worlds:

- [Laravel](https://laravel.com/) - the leading PHP framework
- [Filament](https://filamentphp.com/) - the leading RAD framework for Laravel
- [Moox](https://moox.org/) - my own ecosystem of Laravel and Filament packages
- [WordPress](https://wordpress.org/) - the CMS and Block editor behind Moox
- [React](https://react.dev/) - powering the interactive experience
- [ThreeJS](https://threejs.org/) - entering the third dimension
- [React Three Fiber](https://github.com/pmndrs/react-three-fiber) - React and Three together
- [Motion](https://motion.dev/) - the animation framework
- [Drei](https://github.com/pmndrs/drei) - Ready made concepts for React Three Fiber
- [TailwindCSS](https://tailwindcss.com/) - the very best CSS framework

As well as [Nginx](https://nginx.org/), [PHP](https://www.php.net/) and [MySQL](https://www.mysql.com/) on [Linux](https://www.linux.org/), deployed with [Laravel Forge](https://laravel.com/forge) on [Hetzner Cloud](https://www.hetzner.com/cloud/).
