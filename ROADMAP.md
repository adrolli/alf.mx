## Features

- **Startpage**  
  A keyed video of myself sitting with a MacBook.  
  Subtle micro-interactions (typing, facial movement).  
  When scrolling, I react slightly — as if noticing the visitor.

- **Interactive timeline ("History")**  
  Each scene is a desk setup representing a specific year.  
  Hardware and tools evolve over time.

- **Object interactions (tooltips)**
  - _Call me_ → opens contact modal / phone link
  - _Explore_ → shows project details
  - _Listen_ → controls audio player
  - _Look_ → opens image gallery
  - _Switch_ → toggles dark mode / scene lighting

- **Audio experience**  
  Music and radio evolve with the timeline.  
  Transitions are controlled by scroll — fading between decades.

- **Future section**  
  A shift from physical interfaces to spatial interaction:  
  VR + AI driven experience.

- **Languages**
  - English
  - German  
    (audio content currently available in these two)
