# Moox Wayback

A Wayback Machine / Archive.org Downloader made with Laravel.

## Features

<!--features-->

- Artisan Command `moox:wayback` restores websites from Archive.org
- Combines the results from multiple snapshots to get a fully working site
- Resource for Filament to manage restored websites with Wayback action

<!--/features-->

## Requirements

See [Requirements](https://github.com/mooxphp/moox/blob/main/docs/Requirements.md).

## Installation

```bash
composer require moox/wayback
php artisan moox:install
```

Curious what the install command does? See [Installation](https://github.com/mooxphp/moox/blob/main/docs/Installation.md).
